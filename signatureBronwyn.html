How to Install Your Email Signature

Instructions for Gmail
  1. Open Gmail → Settings ⚙️ → See all settings
  2. Scroll to the Signature section
  3. Click Create New
  4. Click the Copy Signature button above
  5. Paste into Gmail’s signature editor using Ctrl + V
  6. Set it as default and click Save Changes
Instructions for Outlook
  1. Go to File → Options → Mail → Signatures
  2. Click New and name your signature
  3. Click the Copy Signature button above
  4. Paste it into the editor using Ctrl + V
  5. Assign and click OK

Perform this setup on desktop to sync to mobile.
